# 履修計画サンプル

このディレクトリには、AP-Univ-Plannerで読み込める履修計画JSONのサンプルを置きます。

## ファイル

- [`sample_plan.json`](sample_plan.json)
  - 情報システム専攻の履修例です。
  - サンプルとして用意する履修例は、この1件のみです。

## 使い方

1. `sample_plan.json` をダウンロードします。
2. AP-Univ-Plannerを開きます。
3. 画面上部の **読込(.json)** からファイルを選択します。
4. 読み込み後、修了要件進捗や時間割表示を確認します。

## 注意事項

- この履修例は、入力形式や操作感を確認するための例です。
- 情報システム専攻の一例であり、推奨履修モデルや修了保証ではありません。
- 実際の履修登録・修了判定は、必ず大学公式のシラバス、学生便覧、学務システム、学内掲示等で確認してください。
- サンプルJSONを公開する前に、氏名、学籍番号、メールアドレス、研究室名、個人的なメモなどの非公開情報が含まれていないことを確認してください。
