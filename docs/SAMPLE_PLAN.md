# 履修計画サンプル

AP-Univ-Plannerに同梱している履修計画サンプルについて説明します。

## サンプルファイル

- [`examples/sample_plan.json`](../examples/sample_plan.json)

このサンプルは、**情報システム専攻の履修例**です。公開する履修例はこの1件のみです。

## 目的

サンプルでは次を確認できます。

- 保存済み履修計画JSONの形式
- アプリへの読込手順
- 時間割・特殊枠・修了要件進捗の表示例
- 情報システム専攻として読み込んだ場合のカテゴリ表示例

このサンプルは、推奨履修モデルや修了保証ではありません。

## 使い方

1. [`examples/sample_plan.json`](../examples/sample_plan.json) をダウンロードします。
2. AP-Univ-Plannerを開きます。
3. 画面上部の **読込(.json)** を選択します。
4. ダウンロードした `sample_plan.json` を読み込みます。
5. 時間割、特殊枠、修了要件進捗、通知欄を確認します。

読み込み後は、自分の所属・履修計画に合わせて編集してください。

## サンプルの主な内容

- `app`: `ap-univ-planner`
- `appVersion`: サンプル作成時のツール本体バージョン
- `version`: 保存データ形式 `2`
- `myCourse`: `1`（情報システム専攻）
- `state`: M1前期からM2後期までの履修計画

## 注意事項

- 実際の履修登録・修了判定は、必ず大学公式情報で確認してください。
- 年度や公式情報の更新により、サンプルの開講状況が実際と合わなくなる場合があります。
- 自分の計画の土台として使う場合も、読み込み後に内容を確認・編集してください。

サンプルの更新・公開確認を行う管理者は、[`docs/MAINTAINER_GUIDE.md`](MAINTAINER_GUIDE.md) を参照してください。
